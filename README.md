# Smart-Study-Hub
Smart Study Room &amp; Productivity Hub is a web app to boost focus, manage tasks, and improve study productivity using HTML, CSS, and JavaScript.

📌 Features
⏱️ Pomodoro Timer – Focused 25-minute study sessions with start, pause, and reset options
📝 Task Manager – Add, complete, and track daily tasks easily
📒 Quick Notes – Save important notes using local storage
📊 Study Analytics Dashboard – Tracks completed tasks and study progress
🔄 Data Persistence – Notes and data remain saved after page refresh
🎯 Join Study Room – Simulated focused study environment for motivation
🖥️ Modern UI – Clean, responsive design with a dark theme

🛠️ Tech Stack
HTML
CSS
JavaScript

🚀 How to Run
Clone the repository
Open index.html in your browser
Start using the Smart Study Room 🎯
